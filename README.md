# Accurate Cannon Launch

A mod that tweaks the Probe Tracking Module to properly break off from the Orbital Probe Cannon, and fall into Giant's Deep at the start of each loop. 

To give you a better view without use of freecam mods, this mod also allows you to spawn a projection pool at the launch tower connected to the pool aboard the falling module - or spawn the player within Giant's Deep to see the module's descent up-close.

Both of these QOL features can be configured in the mod's settings in-game! Otherwise, you can simply use the signalscope to zoom in and watch from the launch tower.
